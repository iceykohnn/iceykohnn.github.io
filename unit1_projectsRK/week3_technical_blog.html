Let;s look at the enumerable #map. Map and collect are synonymous. This works best with arrays, but can be used with ranges and hashes. Map takes eaxh element in the array/range/hash and iterates it, or looks specifically at it as one element, processes it through the code block we supply, and puts that value into a new array with the processing applied. Remember that the number of items in will always wqual the number of the items out. Map would be most helpful in a situation like when you are trying to perform a function for an entire set of items in an array/range/hash, i.e. trying to capitalize all the names in an array.
